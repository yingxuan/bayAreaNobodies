## C) 核心功能实现

### 1. 一句话解读生成规则（前端规则）
基于 `title` 和 `tags` 关键词匹配：

- **Google / Search** → "搜索入口正在被 AI 重塑"
- **Meta / Ads** → "广告与投放岗位进一步被 AI 压缩"
- **Nvidia / Chips** → "算力仍是核心瓶颈"
- **Binance / Wallet / Security** → "资产安全风险上升，需关注"
- **AI / LLM / GPT** → "AI 技术正在改变行业格局"
- **BigTech** → "大厂战略调整，影响行业走向"
- **Infrastructure** → "基础设施服务正在演进"
- **Career / Layoffs** → "科技行业就业市场出现变化"
- **兜底** → "科技圈出现新的方向变化，值得关注"

### 2. 影响分析生成规则
基于 `tags` 和 `summary` 匹配：

- **Security** → "对你意味着：→ 检查资产安全设置，关注钱包与交易所动态"
- **AI / Career** → "对你意味着：→ 关注 AI 工具使用，考虑技能转型方向"
- **Chips / Infrastructure** → "对你意味着：→ 算力成本可能影响项目预算，关注云服务价格"
- **BigTech** → "对你意味着：→ 大厂动向影响行业趋势，关注相关投资与就业机会"
- **兜底** → "对你意味着：→ 科技趋势变化，值得持续关注"

### 3. 今日重点标记
- 规则：热度最高（score 最高）的一条标记为 🔥 今日重点
- 视觉：`border-2 border-orange-300 bg-orange-50/30`（橙色边框+浅橙色背景）
- 位置：标题前，与分类标签一起显示

### 4. 原帖来源弱化
- 位置：卡片底部，分隔线下方
- 样式：`text-xs text-gray-400`（小字、浅灰色）
- 内容：`🔥 {score} | by {author}`（仅显示分数和作者）

## D) 验收标准

✅ **用户无需读英文即可理解在发生什么**
- 英文原标题完全不显示
- 所有内容都是中文

✅ **每条都有"所以呢"的答案**
- 每条都有"一句话解读"
- 每条都有"对你意味着：→ ..."的影响分析

✅ **第一条一眼看出是"今天最重要"**
- 热度最高的条目有 🔥 今日重点 标记
- 橙色边框+背景，视觉突出

✅ **整个 section 看起来不像 Hacker News**
- 不再显示英文原标题
- 有解读和影响分析，更像"简报"而非"新闻列表"
- 副标题明确："今天值得你花 2 分钟知道的 3 件事"

## E) 技术实现细节

### 前端规则生成（无后端依赖）
- `generateTechSummary()`: 基于关键词匹配生成解读
- `generateTechImpact()`: 基于 tags 和 summary 生成影响分析
- 所有规则都是 deterministic（相同输入产生相同输出）
- 无外部 API 调用，纯前端逻辑

### 样式设计
- 今日重点：`border-2 border-orange-300 bg-orange-50/30`
- 普通卡片：`border border-gray-200`
- 原帖来源：`text-xs text-gray-400` + 分隔线

### 数据流
1. 从 API 获取 3 条科技新闻（已包含 title, tags, score, author）
2. 按 score 排序，找出热度最高的
3. 对每条生成：
   - 中文主标题（`generateChineseTitle`）
   - 一句话解读（`generateTechSummary`）
   - 影响分析（`generateTechImpact`）
4. 渲染卡片，第一条标记为"今日重点"
