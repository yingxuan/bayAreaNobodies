# 「科技圈新动向」模块重构总结

## 问题
- 当前科技卡多为宏观判断（如"正在被重塑""值得关注"）
- 强调趋势但没有明确对象、影响和行动建议
- 湾区码农用户无法在 5 秒内判断"这条值不值得看"

## 解决方案
重构为**三段式结构**，确保每条科技动向都包含：
1. 【发生了什么】（具体到公司/产品/技术）
2. 【对你意味着什么】（湾区码农视角）
3. （可选）【你现在可以做什么】

## 修改的组件/函数

### 新增文件
1. **web/app/lib/techContent.ts** (新增)
   - `extractEntity()`: 从标题提取公司/产品名
   - `generateConcreteTitle()`: 生成具体标题（具体对象 + 明确事件）
   - `generateWhatHappened()`: 生成"发生了什么"（必须包含具体公司/产品/技术）
   - `generateWhatItMeans()`: 生成"对你意味着什么"（湾区码农视角）
   - `generateWhatYouCanDo()`: 生成"你现在可以做什么"（可选）

### 修改文件
2. **web/app/components/TechRadar.tsx** (修改)
   - 移除 `generateTechSummary` 和 `generateTechImpact` 调用
   - 使用新的三段式结构函数
   - 更新 UI 显示：标题 + 发生了什么 + 对你意味着 + 你现在可以

## 禁止抽象用语
以下词语已从主结论中移除：
- ❌ 正在被重塑
- ❌ 出现新变化
- ❌ 值得关注
- ❌ 方向变化
- ❌ 趋势明显

## 标题规则

### ✅ 正确示例
- "Google 搜索正在 AI 化"（具体对象 + 明确事件）
- "NVIDIA 新工具影响 AI 训练成本"（具体对象 + 具体影响）
- "AWS 推出新的云服务"（具体对象 + 具体动作）
- "OpenAI 调整 API 定价"（具体对象 + 具体变化）

### ❌ 错误示例（已修复）
- "AI 热点"（无具体对象）
- "科技圈出现新变化"（抽象用语）
- "值得关注"（无具体事件）
- "方向变化"（无具体对象）

## 验收标准

✅ **用户能在 5 秒内回答：这条和我有什么关系**
- 通过"对你意味着"部分，用户能立即理解影响

✅ **至少能判断：要不要点、要不要继续看**
- 通过"发生了什么"和"对你意味着"，用户能判断相关性

✅ **科技卡不再像"新闻摘要"，而像"个人决策提醒"**
- 三段式结构让每条内容都像个人化的决策建议

## 技术实现

### 实体提取
支持 20+ 主要科技公司：
- 大厂：Google, Meta, Microsoft, Amazon, Apple, Netflix
- AI：OpenAI, Anthropic
- 云服务：AWS, Vercel, GitHub
- 基础设施：NVIDIA, Docker, Kubernetes, Terraform
- 数据库：MongoDB, Redis, Databricks, Snowflake

### 产品/技术提取
- GPT 系列：GPT-3, GPT-4, GPT-5
- AI 模型：Claude, Gemini, ChatGPT
- 技术：搜索、GPU、芯片、云服务、API

### 内容生成规则
- **发生了什么**：必须包含具体公司/产品/技术，禁止抽象用语
- **对你意味着**：从湾区码农视角，关注技能、工作方式、职业路径、Side Project
- **你现在可以**：提供具体行动建议（学/用/关注/忽略），可选

## 数据流

1. 前端从 `/tech/trending` API 获取数据（HN）
2. 对每个 item，调用 `generateConcreteTitle()` 生成标题
3. 调用 `generateWhatHappened()` 生成"发生了什么"
4. 调用 `generateWhatItMeans()` 生成"对你意味着"
5. 调用 `generateWhatYouCanDo()` 生成"你现在可以"（可选）
6. 在 UI 中按三段式结构显示
